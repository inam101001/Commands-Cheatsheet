= 1.2.0Exactly this version
!= 1.2.0Any version except this
> 1.2.0Greater than
>= 1.2.0Greater than or equal
~> 1.2.0Pessimistic: >= 1.2.0, < 1.3.0
~> 1.2Pessimistic: >= 1.2, < 2.0

🔄 Taint, Untaint & Replace
terraform taint/untaint are deprecated since TF 0.15.2. Use -replace flag instead.
Legacy (deprecated)
terraform taint TYPE.NAMEMark resource for forced replacement
terraform taint 'module.vpc.aws_subnet.pub'Taint module resource
terraform taint -allow-missing TYPE.NAMEDon't error if resource not found
terraform untaint TYPE.NAMEClear taint mark from resource
Modern Replacement (preferred)
terraform apply -replace=TYPE.NAMEForce destroy+recreate specific resource
terraform apply -replace=aws_instance.webReplace EC2 instance
terraform plan -replace=TYPE.NAMEPreview replacement without applying
terraform plan -replace=TYPE.A -replace=TYPE.BReplace multiple resources
terraform apply -replace=TYPE.A \Combined with auto-approve
-auto-approve
Lifecycle replace_triggered_by
resource "aws_instance" "web" { lifecycle { replace_triggered_by = [ aws_security_group.web.id ] } }

🚀 Best Practices & Advanced Features (TF 1.x)
moved Block (TF 1.1+)
# Rename resource in state without destroying/recreating moved { from = aws_instance.web to = aws_instance.app_server } # Move into/out of module moved { from = aws_security_group.sg to = module.app.aws_security_group.sg } # For each → for each key change moved { from = aws_instance.web[0] to = aws_instance.web["server-a"] }
import Block (TF 1.5+)
# Declarative import (preferred over CLI import) import { to = aws_s3_bucket.main id = "my-existing-bucket" } # With for_each (TF 1.7+) import { for_each = { foo = "bucket-foo" bar = "bucket-bar" } to = aws_s3_bucket.main[each.key] id = each.value } # Generate config automatically (plan flag) # terraform plan -generate-config-out=generated.tf
check Block (TF 1.5+)

precondition / postcondition (TF 1.2+)
